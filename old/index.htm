equatio

An interactive browser experiment by nucular

You can enter a simple JavaScript expression into the text field below. It will be called for every pixel which will be coloured depending on the result.

Look at the examples and then mess around and have fun!

These variables are predefined:
Pixel position: (x, y) ; Mouse position: (mx, my)
Mouse button state: md ; Mouse position before pressed: (mdx, mdy)
Mouse position delta: (dx, dy) ; Time: t ; Time speed: td
